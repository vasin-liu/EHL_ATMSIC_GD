数据字典维护

数据字典维护主要是针对系统字典表有所增加或变动时，对字典代码进行的操作。如：新增字典信息、编辑字典信息、删除字典代码。

  • 在查询区域选择系统类别、代码类别，输入代码、代码说明点击查询按钮，列表将显示符合条件的数据。

  • 点击工具条的新增将弹出字典信息新增界面，在相应的输入框录入字典信息，点击保存按钮完成字典信息新增。

  • 点击工具条的编辑将弹出字典信息编辑界面，在允许的输入框内重新录入字典信息，点击保存按钮完成字典信息编辑。

  • 在数据列表中选中一条或多条数据后，点击工具条上的删除按钮，提示是否删除记录消息，点击确定完成数据删除。

  • 点击工具条上的刷新按钮可以直接刷新数据列表。
